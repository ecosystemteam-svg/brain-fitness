## 🚀 Deploy to Vercel

### วิธีที่ 1: ผ่าน GitHub (แนะนำ)
1. สร้าง GitHub Repository ใหม่
2. Push โค้ดนี้ขึ้น GitHub:
   ```bash
   git init
   git add .
   git commit -m "Brain Fitness Challenge v4"
   git remote add origin https://github.com/YOUR_USERNAME/brain-fitness.git
   git push -u origin main
   ```
3. ไปที่ [vercel.com](https://vercel.com) → New Project → Import จาก GitHub
4. เลือก repo → Deploy → เสร็จ!

### วิธีที่ 2: Vercel CLI
```bash
npm i -g vercel
vercel
```
